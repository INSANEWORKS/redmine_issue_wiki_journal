## 貢献する

### Pull Request

  1. リポジトリをフォーク
  2. Feature ブランチを作る (``git checkout -b new-feature``)
  3. あなたの変更を commit (``git commit -am ``add some new feature``)
  4. ブランチを push (``git push origin new-feature``)
  5. Pull Request を作成
